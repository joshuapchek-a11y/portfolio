THE BUREAU

The Bureau is a project responding to the Chance and Choice worldbuilding brief. The project takes the form of an interactive game built on Figma, where the user plays as a character starting their new job as a design approver in a world with flat temporality for objects - where any single design approval could have far reaching consequences. The design features a custom UI design styled after Windows 95, and visuals which were originally collaged, before being styled by Adobe Firefly.
